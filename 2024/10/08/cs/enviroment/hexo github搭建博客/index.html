hexo github搭建博客

发表于 2024-10-08 更新于 2025-12-17 分类于 环境 ， hexo

系统debain12 nodejs 20.18.0 npm 10.8.2

安装nodejs

因为apt安装的nodejs版本低于hexo要求，所以用预编译版本安装

node
下载后重命名为nodejs 并解压到/usr/local/lib下
创建连接

执行node确认后安装hexo客户端

然后为hexo创建连接

创建blog并安装hexo-deployer-git插件

hexo s在本地打开服务

部署到github

本地准备ssh
  • 登陆github创建仓库，名为name.github.io
  • 点击右上角头像->settings. 左边栏找到 SSH and GPG keys 点add ssh keys
  • 将~/.ssh/githubkey.pub的内容全部复制到那个大框里，命名再点add SSH key

在本地终端

连接成功则没问题

编辑blog下的_config.yml

/.ssh/config里指定密钥为/.ssh/github

参考文档中的一键部署hexo

访问名字.github.io
