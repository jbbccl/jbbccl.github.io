less-46 ORDER BY后

ORDER BY类似给每行加序号
ORDER BY 1 指使用第一列(id)作为每行的序号
ORDER by '1' 每行序号都是'1'没有区别，所以使用默认排序
ORDER BY if(id=10,0,1) 仅当id=10时序号为0，其他行都为1，所以id=10在最前面
