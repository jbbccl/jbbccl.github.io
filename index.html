hexo github搭建博客

系统unbuntu22.4.4（虚拟机） nodejs 20.18.0 npm 10.8.2

安装nodejs

因为apt安装的nodejs版本低于hexo要求，所以用预编译版本安装

node
下载后重命名为nodejs 并解压到/usr/local/lib下
创建连接

执行node确认后安装hexo客户端

然后为hexo创建连接

创建blog并安装hexo-deployer-git插件

hexo s在本地打开服务

部署到github

本地准备ssh
  • 登陆github创建仓库，名为name.github.io
  • 点击设置，找到 SSH and GPG keys 点add ssh keys
  • 将~/.ssh/githubkey.pub的内容全部复制到那个大框里，命名再点add SSH key

在本地终端

连接成功则没问题

编辑blog下的_config.yml

参考文档中的一键部署hexo

这里报错的话在

访问名字.github.io

…需要一段时间才会更新

虚拟机安装ubuntu22.04

win10下 虚拟机安装ubuntu22.04

下载vm下载

  • 打开vm17安装包安装
  • 选择个人版，添加虚拟机，选自定义，选择ubuntu…..iso
  • 由于vm强制简易安装 ，创建完成后要先退出安装，再安装以便设置中文

安装完成，菜单里-虚拟机-拍摄快照
